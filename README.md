# MedicAlarm-APP

Aplicación móvil para el proyecto MedicAlarm de la materia Desarrollo de un Proyecto de Software

## Please use yarn to install dependencies

```
yarn
```

### Install dependencies

Example:

```
yarn add axios
```

## Run Dev

### Android

```
yarn android
```

### iOS

```
yarn ios
```
